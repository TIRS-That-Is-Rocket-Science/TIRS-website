# TIRS-website
Lieu du dépôt du contenu permettant la construction du website TIRS: thatisrocketscience.com.

Prototype minimal basé sur Astro, MDX, et un composant Three.js (react-three-fiber).

Commandes:

```bash
npm install
npm run dev
```

Deploy: Le workflow `.github/workflows/deploy.yml` build et publie `./dist` sur GitHub Pages.
